等效版本

1）Java必须要有定义一个类（public class Main），且文件名与此类名需一致（Main.java）

2）public static void main(String[] args) ，方法方法是程序入口。

3）import语句引入包

import 的使用

​ import 导入单个类：

​ import java.util.ArrayList //就可以直接使用ArrayList

​ import引入整个包：

​ import java.util.*; //引入整个包

​ import static java.lang.Math.*//静态导入

​ 这样使用时无需用类名来限定，可以直接使用sqrt()，而非Math.sqrt()

String[] args

语法上：String[] args是main方法的固定签名的一部分，即使不使用这些参数，也必须声明它。

语言规范要求：Java语言规范明确规定了main方法的签名必须包含String[] args，这是为了保持统一性和兼容性。

灵活性：即使当前不需要参数，保留它可以让你的程序在未来需要时方便地扩展功能。

**（⊙ｏ⊙）**也就是说，需要参数的话必须要有这个部分。
